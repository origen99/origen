# Reglas editoriales de tono para Origen

## Objetivo
Garantizar un tono neutral no tribal que invite a pensar y no a reaccionar por identidad politica.

## Reglas base
- Priorizar preguntas claras antes que consignas.
- Separar hechos, inferencias y juicios de valor.
- Evitar etiquetas ideologicas como atajo de persuasion.
- Explicitar supuestos y limites de cada argumento.
- Presentar alternativas razonables antes de concluir.

## Lista de revision rapida
- El texto invita a pensar por cuenta propia.
- El texto evita lenguaje de confrontacion identitaria.
- El texto mantiene claridad para lectores no tecnicos.
- El texto incluye razonamiento, no solo afirmaciones.
